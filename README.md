Rock Paper Scissors

A simple Rock–Paper–Scissors web game built with HTML, CSS, and JavaScript.
The game gives instant visual feedback:

Green when you win

Red when you lose

Gray when it’s a draw



Technologies used:
HTML
CSS
JAVASCRIPT



Instruction how to run the game:

1. Download or clone the project files

2. Make sure all files are in the same folder

3. Open index.html using any modern web browser

4. Click Rock, Paper, or Scissors to start playing
